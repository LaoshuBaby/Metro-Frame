## Metro-Frame
Maybe the best Metro application in the World

## 简体中文版说明（zh_CN）

该项目由中国各地的地铁迷和程序员进行维护

欢迎您来为这个世界上最大最全的地铁应用添砖加瓦

## 感谢（Acknowledge）

为本项目提供产品层次上的指导的威尔先生

BUCT计科系的几位努力敲代码的学长们

肯为本项目开一个接口，作为MF的在线地图引擎之一的先生

百度地图开放的API使我们获取了世界上一百多个城市的地铁信息（当然还是有待完善的）

以及全国各地的积极校对自己城市内地铁信息的地铁迷们。正是因为有了你们，才能够发展完善
